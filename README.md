# learning-resource

此仓库建立的目的：

1. 整理过去学习过的资料
2. 审视自身的学习方式与方法
3. 发现学习当中产生的问题
4. 不断的反思与总结
5. 提高学习效率
6. 合理的安排学习时间
7. 系统的构建知识体系
